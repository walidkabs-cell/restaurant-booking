# Restaurant Booking System

A modern restaurant reservation system with customer booking and admin dashboard.

## Features

- **Customer View**: Browse tables, make reservations
- **Admin Dashboard**: View all bookings, filter by date/status, manage reservations
- **SQLite Database**: Persistent storage with better-sqlite3
- **Responsive Design**: Works on mobile and desktop

## Setup

```bash
npm install
npm start
```

Server runs on port 3000 by default.

## URLs

- **Customer**: http://localhost:3000
- **Admin**: http://localhost:3000/admin

## Tech Stack

- Node.js + Express
- SQLite (better-sqlite3)
- Vanilla JavaScript (no framework)
